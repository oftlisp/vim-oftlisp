# vim-oftlisp

Syntax highlighting for OftLisp in Vim and Neovim.

## Installation

### Plug

```vim
call plug#begin()

" ...

Plug 'oftlisp/vim-oftlisp'

" ...

call plug#end()
```

### Dein

```vim
if dein#load_state('...')
  call dein#begin('...')

  " ...

  call dein#add('oftlisp/vim-oftlisp')

  " ...

  call dein#end()
  call dein#save_state()
endif
```
